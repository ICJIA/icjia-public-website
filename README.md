# Installation

## Clone repository

```
git clone https://github.com/ICJIA/icjia-public-website.git
```

## Install Grunt:

```
npm install -g grunt-cli
```

## Install dependencies

In  ```_themes/icjia/```:

```
npm Install
```

# Run Statamic server check:

```
http://<your-host>/check/check.
```

Make sure ```mod_rewrite``` is installed.
