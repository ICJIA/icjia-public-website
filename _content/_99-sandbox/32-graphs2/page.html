Responsive CSS Triangles

Triangles that can be adjusted by changing their "w" (width percentage) and "h" (height percentage) to be responsive in relation to their parent container.

- You can set a fixed height to the parent container and the height property will be proportionate to that fixed height.
- If you'd like an equilateral triangle you can set both the "w" and "h" to be the same value or just leave one of them blank.
