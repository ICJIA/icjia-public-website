A) Overview - top

Git CSS3 Buttons helps you easily create GitHub-style buttons from links, buttons, and inputs. Git CSS3 Buttons has been designed to be as easy and flexible as possible.

B) CSS Files and Structure - top

All you need to do is link the gitbuttons.css file.

<link rel="stylesheet" href="gitbuttons.css" type="text/css">

C) Browser Compatibility - top

Firefox 3+, Google Chrome, Safari 4+, IE 6+, Opera 10+.

Note: Some CSS3 features are not supported in older versions of Opera and versions of Internet Explorer prior to IE 8. Also there are a few position issues with ie6

D) How to use - top

It is very easy to learn how to use Git CSS3 Buttons. All you need to do is add at least class="gbutton" to either a link, button, or input.

Standard Buttons - top

The "buttons" can be created by adding class="gbutton" to any appropriate <a>, <button>, or <input> element.

<a href="#" class="gbutton">Post comment (link)</a>
<input class="gbutton" type="submit" value="Post comment (input)">
<button class="gbutton" type="submit">Post comment (button)</button>
Post comment (link)

Add a further class of pill to create a button pill-like button.

<a href="#" class="gbutton pill">This is a pill button</a>
This is a pill button

Add a further class of active to make the button active and you can use it as indicator.

<a href="#" class="gbutton active">Publish post</a>
Publish post

Buttons with positive actions - top

If you have a button that needs to indicate a visual positive action, like funds being available, this can be indicated by adding the class positive.

<a href="#" class="gbutton positive">Funds Available</a>
Funds Available

You can add the class active on to the end of any color style such as positive, negative, black, orange, or any style you create with the style creator.

<a href="#" class="gbutton positive active">Funds Available</a>
Funds Available

Buttons with negative actions - top

If you have a button that needs to indicate a visual negative action, like funds not being available, this can be indicated by adding the class negative.

<a href="#" class="gbutton negative">Funds Not Available</a>
Funds Not Available

You can add the class active on to the end of any color style such as positive, negative, black, orange, or any style you create with the style creator.

<a href="#" class="gbutton negative active">Funds Not Available</a>

Grouped buttons - top

Groups of buttons can be created by wrapping them in an element given a class of gbutton-group. A less important group of buttons can be marked out by adding a further class, minor-group.

<div class="gbutton-group">
    <a href="#" class="gbutton bold">Dashboard</a>
    <a href="#" class="gbutton">Inbox</a>
    <a href="#" class="gbutton">Account</a>
    <a href="#" class="gbutton">Logout</a>
</div>
Dashboard Inbox Account Logout 
<ul class="gbutton-group">
    <li><a href="#" class="gbutton bold pill">Dashboard</a></li>
    <li><a href="#" class="gbutton pill">Inbox</a></li>
    <li><a href="#" class="gbutton pill">Account</a></li>
    <li><a href="#" class="gbutton pill">Logout</a></li>
</ul>

Mixed groups - top

Displaying a mixture of grouped and standalone buttons, as might be seen in a toolbar, can be done by adding another wrapping element with the class gbutton-container.

<div class="actions gbutton-container">
    <a href="#" class="gbutton bold">Compose new</a>
    
    <div class="gbutton-group">
        <a href="#" class="gbutton bold">Archive</a>
        <a href="#" class="gbutton">Report spam</a>
        <a href="#" class="gbutton negative">Delete</a>
    </div>
                        
    <div class="gbutton-group minor-group">
        <a href="#" class="gbutton">Move to</a>
        <a href="#" class="gbutton">Labels</a>
    </div>
</div>
Compose new
Archive Report spam Delete
Move to Labels

Control Panel buttons - top

You can create groups of buttons or individual buttons that can give indications such as, accounts paid or if a user has downloaded a product

<div class="gbutton-group">
    <a href="#" class="gbutton large bold">User Id 64<</a>
    <a href="#" class="gbutton large bold positive active">Acccount Paid</a>
    <a href="#" class="gbutton large bold positive active">App Downloaded</a>
    <a href="#" class="gbutton large bold negative active">User Logged in</a>
</div>
User id: 64 Acccount Paid App Downloaded User Logged in

Mixed groups can be used effectively to put indication and control in the same button container

<div class="actions gbutton-container">
    <a href="#" class="gbutton bold">User id: 64</a>
    
    <div class="gbutton-group">
        <a href="#" class="gbutton positive active">Account Paid</a>
        <a href="#" class="gbutton positive active">App Downloaded</a>
        <a href="#" class="gbutton negative active">User Logged in</a>
    </div>
                        
    <div class="gbutton-group minor-group">
        <a href="#" class="gbutton">Edit User</a>
        <a href="#" class="gbutton negative">Delete User</a>
    </div>
</div>
